# Digital Clock

A simple Digital Clock made using HTML, CSS and JavaScript.

This project displays the current time and updates it every second using JavaScript. It helped me practice DOM manipulation, JavaScript Date object and setInterval().

## Features

* Shows current system time
* Updates automatically every second
* Clean and simple user interface
* Responsive centered layout
* Neon glow clock design

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
Digital-Clock/
│
├── index.html
├── style.css
└── first.js
```

## How It Works

* JavaScript gets the current time using the Date object.
* The time is converted into a readable format using `toLocaleTimeString()`.
* `setInterval()` updates the clock every second.
* The time is displayed inside the clock container using DOM manipulation.
